# 🏷️ 명찰 자동 생성기 (Nameplate Creator)

인공지능학과 명찰을 엑셀 명단으로부터 자동 생성하는 Python 스크립트입니다.

## ✨ 주요 기능

- **엑셀 → PPT 자동 변환**: `.xlsx` 파일에서 이름을 읽어 PowerPoint 명찰을 자동 생성
- **템플릿 기반 디자인**: 기존 PPT 템플릿(`명찰 디자인.pptx`)의 디자인과 스타일을 그대로 유지
- **슬라이드 자동 복제**: 이름 수에 따라 슬라이드를 자동으로 복제 (슬라이드 당 4명)
- **서식 보존**: 원본 텍스트의 폰트, 크기, 굵기, 기울임 등 스타일을 유지

## 📋 사전 준비

### 필요 환경
- Python 3.x

### 패키지 설치

```bash
pip install openpyxl python-pptx
```

### 필요 파일

| 파일 | 설명 |
|------|------|
| `명찰 디자인.pptx` | 명찰 템플릿 (플레이스홀더: `이름1`, `이름2`, `이름3`, `이름4`) |
| 명단 엑셀 파일 (`.xlsx`) | A열에 이름 목록 (2행부터 시작) |

## 🚀 사용법

```bash
python createNameplate.py <명단.xlsx>
```

### 예시

```bash
python createNameplate.py 학생명단.xlsx
```

실행하면 동일 폴더에 `완성_명찰.pptx` 파일이 생성됩니다.

## 📁 프로젝트 구조

```
.
├── createNameplate.py    # 메인 스크립트
├── 명찰 디자인.pptx       # 명찰 PPT 템플릿
├── 완성_명찰.pptx         # 생성된 결과물 (실행 후 생성)
└── README.md
```

## ⚙️ 동작 방식

1. 엑셀 파일의 A열에서 이름 목록을 읽어옵니다 (2행부터)
2. 이름을 4명씩 그룹으로 나눕니다
3. 템플릿 슬라이드를 필요한 만큼 복제합니다
4. 각 슬라이드의 `이름1`~`이름4` 플레이스홀더를 실제 이름으로 치환합니다
5. `완성_명찰.pptx`로 저장합니다
